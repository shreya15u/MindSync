# MindSync

MindSync is an offline-first iOS application designed to help users manage mental overload through voice-based task capture and intelligent task organization. The application enables users to quickly record thoughts, categorize tasks, and stay focused with a distraction-free productivity experience.

## Features

- Voice-based task capture using Apple's Speech Framework
- Speech-to-text conversion
- Task categorization (Urgent, Important, Remember)
- MonoFocus mode for distraction-free task management
- Customizable local reminders using UserNotifications
- Clean and responsive SwiftUI interface
- Offline-first application

## Tech Stack

- Swift
- SwiftUI
- Xcode
- Speech Framework
- UserNotifications
- MVVM

## Installation

1. Clone the repository.

```bash
git clone https://github.com/shreya15u/MindSync.git
```

2. Open the project in Xcode.

3. Build and run the application on an iOS Simulator or a physical device.
